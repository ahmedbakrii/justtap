# Justtap MVP Website

موقع متكامل لبراند المنتجات الرقمية الذكية Justtap المستوحى من تصميم شركة Apple وبشكل بسيط جداً (Minimal).

## 🚀 طريقة التشغيل والرفع على GitHub Pages
1. قم بإنشاء مستودع (Repository) جديد على حسابك في GitHub وسمّه `justtap`.
2. ارفع كافة هذه الملفات مع الحفاظ التام على هيكلية المجلدات المذكورة.
3. توجه لإعدادات المستودع (Settings) ثم إلى قسم Pages.
4. اختر الفرع الرئيسي `main` أو `master` كمصدر للرفع، ثم اضغط حفظ (Save).
5. سيعطيك جيتهاب رابطاً مباشراً ومجانياً لمتجرك ليعمل فوراً أمام زبائنك وبأعلى سرعة وسلاسة!
